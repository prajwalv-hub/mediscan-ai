# MediScan AI 🏥

> AI-Powered Medical Diagnostic Platform

## 🌟 Features

- **📸 Live Camera Scan** — Real-time skin condition scanning using device camera
- **🗣️ Voice Symptom Input** — Speak symptoms in English, Hindi, Telugu, Tamil & more
- **🌍 Multi-Language Support** — AI responds in 6+ Indian regional languages
- **🧬 Multi-Modal Cross-Validation** — Combines image + voice + text for higher accuracy
- **📊 AI Health Risk Score** — Animated 0-100 health gauge with breakdown
- **💬 AI Doctor Chat** — Ask follow-up questions about your diagnosis
- **📄 PDF Report Generation** — Download professional medical reports
- **🩸 Blood Donor Network** — Find nearby donors on interactive map
- **📑 Smart Report Analyzer** — Upload medical reports for lifestyle recommendations
- **🚨 Emergency SOS** — Auto-detection of critical conditions with emergency alerts
- **♿ Accessibility** — Text-to-Speech, High Contrast, Large Text modes

## 🛠️ Tech Stack

- **Frontend**: HTML5, CSS3, Vanilla JavaScript
- **AI Engine**: Google Gemini Pro (Multimodal)
- **Maps**: Leaflet.js (OpenStreetMap)
- **PDF**: jsPDF
- **Camera**: MediaDevices Web API
- **Voice**: Web Speech API (Speech Recognition + Speech Synthesis)
- **Storage**: Browser LocalStorage
- **Deployment**: Vercel
